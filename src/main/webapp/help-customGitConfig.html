Each line provided here will be run with git config --file $WORKSPACE/.config/git/config name value.

Any single-valued variable set here will be overwritten by whatever is in ~/.gitcofig.

An example

        user.name uesr
        user.email user@mydomain
        url.git://myhost.mydomain/git-repo.insteadof https://gerrit.googlesource.com/git-repo
